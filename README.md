# Modern Web Calculator

A stunning, glassmorphism-style calculator built with HTML, CSS, and JavaScript.

## How to Run

### Open in Browser (Recommended)
Simply double-click the `index.html` file in your file explorer, or run the following command in your terminal:

```powershell
start index.html
```

### Python Version (CLI)
If you'd like to run the original Python command-line version:

```powershell
python calculator.py
```
